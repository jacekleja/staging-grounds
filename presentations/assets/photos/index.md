## Usage Guide

| Need | Use |
|------|-----|
| Half-image content slides | `slide111_*` (people + product overlays) or `slide113_*` (pure portraits) |
| Product deep-dive / feature slides | `slide112_*` (UI screenshots, dashboards) |
| Speaker / team slides | `slide113_*` (professional portraits) |
| Before/after screenshots | `slide112_*` (product screenshots) |

## Notes

- All images are ~1060x1200px PNG (portrait orientation, as shown in deck thumbnail grid)
- Slide 111 images: ~1.2-1.8MB each (high-res photos with overlays)
- Slide 112 images: ~68KB-1.3MB (mixed - screenshots are smaller, photos larger)
- Slide 113 images: ~1.4-2.1MB each (high-res lifestyle photos)
- Total: 72 images, ~85MB
